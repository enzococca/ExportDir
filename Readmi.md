# Directory To Excel And PDF Converter

This is a Python application that allows you to export the structure of a directory to Excel and PDF files. 

## Installation

You need to have Python installed on your machine. You can download it from the official website. After installing Python, you can install the required packages using pip:

```bash
pip install PyQt5 openpyxl comtypes
```

## Usage

You can run the application by executing the `export_dir.py` script:

```bash
python export_dir.py
```

The application will show a splash screen and then a welcome message. You will be asked to select a root directory and then save two output PDF files.
